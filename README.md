# H5 魔塔样板

## 简介

HTML5 canvas制作的魔塔样板，支持全平台；即使不会编程的用户，按照模板和说明文档也能很快做出一个魔塔游戏；对于有编程基础的用户，那就更好了，根据文档能做出更个性化的魔塔。
